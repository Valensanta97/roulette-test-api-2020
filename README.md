# README #

This repository contains the Clean Code trial development for the massive company,
where it is developed in the Sprint boot technology with redis.

### Endpoints ###

1. Endpoint de creación de nuevas ruletas : http://localhost:8080/roulettes
2. Endpoint de apertura de ruleta : http://localhost:8080/roulettes/:idRoulette/open
3. Endpoint de apuesta : http://localhost:8080/bet/:idRoulette
4. Endpoint de cierre apuestas : http://localhost:8080/roulettes/:idRoulette/closed
5. Endpoint de listado de ruletas creadas: http://localhost:8080/roulettes
